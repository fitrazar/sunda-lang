### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fitrazar/sunda-lang.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run file
   ```sh
   node sir.js examples/var.sir
   ```
